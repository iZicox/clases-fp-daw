# zillow

1. listado de productos
El listado lo hace muy intuitivo con un mapa a la izquierda y una galeria a la dercha para que el usuario vea la ubicacion de las casas mientras hace su busqueda

2. las caracteristicas de un producto.
Antes de entrar en una casa nos muestra el precio por el numero de habitaciones en caso de rentas y en el casod de ventas nos muestra los datos mas importantes como habitaciones, baños, precio, area y localizacion
Una vez dentro podemos ver mas informacion util de cada propiedad, una descripcion de esta e imagenes para que el usuario entienda mejor.

3. navegación
La navegacion con lo mas importante, venta, renta, hipoteca, agentes que son las principales dudas que tiene una persona al entrar a una pagina de este tipo

4. retorno a home
En cualquiera de los casos tenemos arriba el logo en el centro de zillow que sin importar donde estemos le podemos dar clic para regresar a home

5. migas
Son bantantes intuitivas ya que cuando entramos a una propiedad el enlace es el nombre de la misma

6. filtros sobre los productos del listado
Los filtros cumplen con todo lo que podemos pedir para una pagina de este estilo, ver rentas, ventas, numero de habitaciones, tipos de casas, area, etc.

7. ordenación del listado
Podemos ordenar como queramos, por precio, habitaciones, area, precio, etc.

8. busqueda de un producto
Con solo colocar el codigo postal, ciudad o distrito podemos iniciar la busqueda de las propiedades de la zona
